HTML Table Cheat Sheet



Table Tags

Name Tag Description
Table <Table> Master tag that contains all elements within a table, including headers, columns, and rows.
Table Head <thead> This groups all the heading content in a table.
Table Header <th> Defines specific cells as header cells in a table.
Table Body <tbody> This tag groups all the body content of the table.
Table Data <td> This defines a cell in the table that contains data.
Table Row <tr> This defines a new row in the table.
Table Foot <tfoot> This tag groups all the footer content in the table.

Table Attributes

Name Attribute Description
Colspan colspan This attribute defines the number of colums that a cell should span across.
Rowspan rowspan This attribute defines the number of rows that a cell should span across.
